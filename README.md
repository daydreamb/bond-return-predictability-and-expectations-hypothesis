# bond-return-predictability-and-expectations-hypothesis
This project evaluates the Expectations Hypothesis of the term structure using zero-coupon yield data. I analyze whether forward-spot spreads forecast excess bond returns and yield changes, following Fama and Bliss (1987).
The project further examines out-of-sample forecasting performance in a recursive real-time setting.

Methodology

Construction of zero-coupon yields and forward rates

Computation of one-year holding period returns and excess returns

Forecasting regressions:

Forward-spot spreads predicting excess returns

Forward-spot spreads predicting yield changes

Newey-West adjusted inference

Recursive out-of-sample estimation

Comparison against:

Random walk model

Pure expectations hypothesis

Historical mean benchmark

Key Research Questions

Does the forward-spot spread forecast bond excess returns?

Is the Expectations Hypothesis supported in the data?

How stable are coefficients over time?

Do predictive relations hold out-of-sample?

Extensions

Time-varying risk premia interpretation

Regime-based estimation

Rolling window stability analysis
